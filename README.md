# csc3350-program-timer

## Description
Implement a console-based application. Your task is to implement a program that starts a second program. The main program should compute the execution time of the second program. The second programâ€™s implementation is  up to you.

## Running
1. Open terminal and cd to the location of the main.cpp
2. Compile using 'g++ main.cpp -o main.out'
3. Run using './main.out'
4. Enter name of external program (must be .cpp file)
5. End of program

## Test File Requirements
Test file must be .cpp - 'apple.cpp' file is an example test file.
